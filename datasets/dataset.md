# Dataset Information

This project uses a combined MRI brain tumor dataset collected from multiple publicly available datasets commonly used in medical imaging research.

Datasets used:
- Figshare Brain Tumor Dataset: https://www.kaggle.com/datasets/ashkhagan/figshare-brain-tumor-dataset
- Sartaj Brain MRI Dataset: https://www.kaggle.com/datasets/sartajbhuvaji/brain-tumor-classification-mri
- BR35H Dataset: https://www.kaggle.com/datasets/ahmedhamada0/brain-tumor-detection

Combined Dataset Statistics:
- Total Images: 7,023
- Classes:
  - Glioma
  - Meningioma
  - Pituitary Tumor
  - No Tumor

Dataset Split:
- Training Images: 5,618
- Testing Images: 1,405

All MRI images were preprocessed and resized to 224×224 resolution before model training.

Due to dataset size and licensing considerations, the complete dataset is not directly included in this repository.
